# Decoda v5.1.0 #

A lightweight lexical string parser for BBCode styled markup packaged into a CakePHP plugin and helper.

## Compatibility ##

* v3.x - CakePHP 1.3, PHP 5.2
* v4.x - CakePHP 2.0, PHP 5.2
* v5.x - CakePHP 2.0, PHP 5.3
